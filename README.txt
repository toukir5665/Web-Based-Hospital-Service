Web-Based Hospital Service System
--------------------------------
Run instructions:
 1. Install XAMPP and start Apache + MySQL.
 2. Create a database named `hospital_db`.
 3. Import the SQL file `database.sql` provided.
 4. Place the project folder in your XAMPP/htdocs directory or configure virtual host.
 5. Open http://localhost/hospital_system/index.php
Default admin passcode: CHANGE_THIS_PASSCODE in config.php
